# 拉勾网职位隐藏脚本

本脚本用于隐藏拉勾网搜索列表中你不想看到的某个职位或某个公司发布的所有职位。

本脚本仅适用于职位搜索页面，即地址以 `https://www.lagou.com/jobs/list_` 开头的页面。

若要恢复所有已屏蔽内容请按F12进入console面板输入 `localStorage.removeItem('__lgTools__')` 后回车，然后按F5刷新页面即可。
